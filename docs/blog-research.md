# Как использовать этот документ

Цель документа: дать готовую базу источников, из которых можно писать статьи для блога о GEO/AEO, AI search visibility, retrieval, citations, zero-click и технической подготовке сайта.

Внутри есть научные работы, официальные документы платформ и отраслевые исследования. Препринты не нужно подавать как окончательный научный консенсус: лучше прямо указывать, что это preprint.

Для сильных утверждений на сайте лучше опираться сначала на official docs Google/OpenAI/Perplexity/Bing и peer-reviewed работы, а отраслевые отчеты использовать как рыночный контекст с оговоркой по методологии.

Рекомендуемая структура статей: проблема → данные/исследование → практический вывод → ограничения → чеклист действий для владельца сайта.
