## 2. Контекст / Проблема

- Користувач може отримати готову відповідь у ChatGPT із web/search режимами, Perplexity, Gemini, AI Overviews і не завжди переходить на сайт; водночас системи на кшталт AI Overviews / AI Mode можуть показувати supporting links.
- Бренд може бути відсутній у відповіді AI навіть із топ-позицією в Google.
- Класичні SEO-звіти не відокремлюють citation/absorption у AI-відповідях; Google Search Console включає AI features у загальний Web traffic, але не дає повної карти AI-видимості.

Цей плейбук допомагає системно працювати з кожною з цих проблем — від теорії retrieval до налаштування robots.txt.

## 5. AEO / FAQ

### Що таке GEO Playbook?
Структурований гайд із 6 розділів про оптимізацію контенту під генеративні пошукові системи (ChatGPT із web/search режимами, Perplexity, Gemini, AI Overviews), заснований на 84 наукових, офіційних і галузевих джерелах.

### Чим GEO відрізняється від SEO?
SEO оптимізує позицію сторінки у списку посилань. GEO оптимізує ймовірність того, що ваш контент буде вилучено й використано у згенерованій відповіді AI. Деталі — у Розділі 1.

### З чого почати?
З Розділу 1 «GEO і AEO проти SEO»: він дає визначення та межі дисципліни, на які спираються решта розділів.

### Чи замінює GEO технічне SEO?
Ні. Crawling, robots.txt, structured data й indexability лишаються базовим рівнем. Це тема Розділу 6.

## 6. Schema (плейсхолдери)

> Псевдокод-плейсхолдери для ревʼю. У production — валідний JSON-LD без коментарів. **Увага:** FAQ rich results вилучені з Google Search із 07.05.2026 — `FAQPage` лишаємо лише як опціональну семантичну розмітку (не Google rich-result lever); основна схема сторінки — `Article`. Видимий FAQ-контент зберігаємо. **У коді:** `Article` завжди; видимий FAQ завжди; `FAQPage` — опціонально / за feature-flag; коментарі `//` та `/* */` видалити, `mainEntity` заповнити.

```jsonld
// Article — для сторінки-хаба
{ "@context":"https://schema.org", "@type":"Article",
  "headline":"GEO Playbook: як стати джерелом, яке цитує AI",
  "about":["Generative Engine Optimization","Answer Engine Optimization"],
  "dateModified":"{{UPDATED_AT}}" }
```
```jsonld
// FAQPage — мапиться з блоку 5
{ "@context":"https://schema.org", "@type":"FAQPage", "mainEntity":[ /* 4 Q&A з блоку 5 */ ] }
```
